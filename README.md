# zooniverse_analysis
code to analyze zooniverse labels
readposteriors.ipynb demonstrates how to read posterior probabilities, work with them, and plot them.


[o1_zooniverse_analysis_fbb_processclassifications.py](https://github.com/fedhere/zooniverse_analysis/blob/main/o1_zooniverse_analysis_fbb_processclassifications.py) creates the zooniverse file from the initial classifications data export

[o2_zooniverse_analysis_fbb_groundtruth.py](https://github.com/fedhere/zooniverse_analysis/blob/main/o2_zooniverse_analysis_fbb_groundtruth.py) builds the groundtruth tables

[o3_zooniverse_analysis_fbb_getposteriors.py](https://github.com/fedhere/zooniverse_analysis/blob/main/o3_zooniverse_analysis_fbb_getposteriors.py) generates the posterior probabilities
